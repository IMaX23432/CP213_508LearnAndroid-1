AI Auto Game Player (Summary)
Vision
แอพ AI ที่ เล่นเกมแทนมนุษย์ได้อัตโนมัติ
เรียนรู้จากการเล่นของผู้ใช้ โดยไม่ต้องเข้าเกมเอง

Problem
ไม่มีเวลาเล่นเกมซ้ำๆ
ระบบออโต้ในเกมจำกัด
บอททั่วไปไม่ยืดหยุ่น เสี่ยงโดนแบน

Solution
ผู้ใช้เล่นเกมผ่านแอพ → ระบบบันทึกพฤติกรรม → ออโต้เล่นแทน

MVP
รองรับ 1 เกม / 1 ประเภท
เกมแนว Idle หรือ Turn-based
บันทึกการเล่น + Auto-play ได้จริง

Core Features
เล่นเกมผ่านแอพ
บันทึก tap / swipe / timing
โหมดออโต้
ตั้งเวลา / จำนวนรอบ

Next Phase
เรียนรู้สไตล์ผู้เล่น
วิเคราะห์หน้าจอเกม
รองรับหลายประเภทเกม

Key Risk
ผิด ToS เกม
เกมอัปเดตระบบพัง

Pitch
AI ที่เรียนรู้จากมนุษย์และเล่นเกมแทนมนุษย์ได้
