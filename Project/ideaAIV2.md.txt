AI Turn-Based Game Assistant (ToS-Safe)
Vision
AI ผู้ช่วยสำหรับเกม Turn-based
ช่วยคิด วิเคราะห์ และแนะนำการตัดสินใจในแต่ละเทิร์น
โดยผู้เล่นเป็นคนกดเองทั้งหมด

Why Turn-Based
ไม่ต้อง real-time
วิเคราะห์ได้ละเอียด
ผู้เล่นมีเวลาตัดสินใจ
ไม่ผิด ToS (ไม่ automation)

Problem
ตัวเลือกในแต่ละเทิร์นเยอะ
ต้องจำ skill / buff / cooldown
มือใหม่ตัดสินใจพลาดบ่อย

Solution
AI วิเคราะห์สถานะเกมในแต่ละเทิร์น
แล้วแนะนำ ทางเลือกที่เหมาะที่สุด
“ควรทำอะไรในเทิร์นนี้”

MVP
รองรับ 1 เกม Turn-based
วิเคราะห์จาก Screenshot / Game State
แนะนำ Action ต่อเทิร์น

Core Features
อ่านสถานะ:
HP / Mana
Buff / Debuff
Cooldown

AI Suggest:
ใช้สกิลไหน
โจมตี / ป้องกัน / รอ
เรียนรู้สไตล์ผู้เล่น

Next Phase
วางแผนล่วงหน้า (2–3 เทิร์น)
Build / Team Recommendation
Meta Strategy จากข้อมูลผู้เล่นรวม

Key Benefit
ปลอดภัย ไม่โดนแบน
ช่วยคิดจริง ไม่โกง
ใช้ได้กับหลายเกม Turn-based

Pitch
AI ผู้ช่วยตัดสินใจสำหรับเกม Turn-based ที่คิดแทนคุณทุกเทิร์น
